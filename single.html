Salesforce - Customized Application in Cloud

Navratri Festival Application

The platform provided by Salesforce.com is used to build customized applications and database in cloud. It is based on a simple point and click concept. This documentation is based on the development of the application using Sales Force platform. The application is used for the management of music and dance festival called as “Navratri”. It includes different objects such as Festival, Bands, Attendees and Stages. Each object consists of fields which represents the object and gives the detailed information about the object. In this document we will include the details of how to create a new application, technical issues faced when creating the application and how to overcome those issues.

Creating an application

The first step after creating an account and logging into the website is to build an app by clicking ‘Add app’ tab on the setup page of the salesforce website.

After clicking the add app tab we get the below screen which creates a Festival object which will be the parent object. It defines few fields to store festival data such as festival start date and end date, time of the festival, name of the festival, etc. A permission set is created for security of the object and few tabs for reports, dashboards, etc. including the Festivals tab as shown below.

Creating new festival record

Now we need to add a new festival record inside the festival object. We create new festival record as Navratri. We can also create, delete and edit any number of records. To view all the records we can view it by clicking it on GO button.
